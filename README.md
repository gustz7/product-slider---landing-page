# 🎧 Product Slider - Landing Page

Uma landing page moderna de produto com slider animado, desenvolvida utilizando HTML, CSS e JavaScript puro.

O projeto simula uma página de apresentação de produto premium, com foco em design moderno, animações suaves e experiência visual imersiva.

---

## 🚀 Preview

🔗 Acesse online:  
https://gustz7.github.io/product-slider---landing-page/
---

## 🛠 Tecnologias Utilizadas

- HTML5
- CSS3 (Animações, Flexbox, Transitions)
- JavaScript (Manipulação de DOM e Slider)
- Git & GitHub

---

## ✨ Funcionalidades

- Slider de produtos com transição animada
- Botões de navegação (Next / Previous)
- Indicadores dinâmicos
- Animações suaves com `transform` e `opacity`
- Efeito hover interativo
- Layout moderno estilo tech

---

## 🎯 Objetivo do Projeto

Este projeto foi desenvolvido com o objetivo de praticar:

- Estruturação de layout responsivo
- Animações com CSS
- Manipulação de classes com JavaScript
- Organização de código front-end
- Versionamento com Git
